Bogosort

Sorting that Rocks

Bogosort is a sorting algorithm. It is not the fastest, in fact it can probably considered the slowest but it rocks becuase everytime you try the Bogosort, you may never get it sorted.

The algorithm for this sort starts with an array of elements. Check the array to see if it is sorted or not. If not, randomize the entire array and check again.

*****Bewarned that arrays greater than 6 may take a very long time*****
